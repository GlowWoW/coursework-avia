# coursework-avia
Курсовая работа. Проектирование приложения на C# для работы с БД (MSSQL)

Проектирование и создание БД для предметной области. Создание приложения для работы с БД

**Предметная область:** дея­тельность компании по продаже авиационных билетов. 
Эта крупная компания занимается продажей билетов на пассажирские рейсы различных авиакомпаний в различных городах мира.
Самолёт следует из пункта А в пункт Б без остановок. Даты вылетов устанавливаются по периодичности вылетов в рейсе. По расписанию вылетов клиент может купить билет на определенный самолет, если присутствуют продажи на данную дату и в самолёте еще остались места. Клиенту выдается билет на самолёт.
